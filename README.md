# 库存表自动化管理系统

一个基于Python的Excel库存表自动化工具,用于自动生成销售清单并管理出库记录。

## 📋 项目简介

本项目为东阳市欧亚金银丝有限公司开发的库存管理自动化工具,主要功能包括:

- ✅ 自动生成序号和日期
- ✅ 自动计算净重(毛重-除皮)
- ✅ 按客户和日期自动生成销售清单
- ✅ 支持两种销售清单模板(简单版和详细版)
- ✅ 自动标记入账状态

## 🚀 快速开始

### 环境要求

- Python 3.7+
- openpyxl 库

### 安装依赖

```bash
pip install openpyxl
```

### 使用方法

1. 将 `improve_inventory.py` 脚本放在与Excel文件同一目录
2. 确保Excel文件名为 `库存tmep.xlsx`
3. 运行脚本:

```bash
python improve_inventory.py
```

4. 脚本会生成 `库存_改进版.xlsx` 文件,包含所有销售清单

## 📊 功能特性

### 1. 自动序号
- 使用Excel公式自动生成连续序号
- 删除行时序号自动调整

### 2. 自动日期填充
- 运行脚本时自动填充空白日期为当天
- 格式: YYYY-MM-DD

### 3. 自动生成销售清单
- 按"出库日期 + 出库对象"分组
- 每组生成两个版本:
  - **简单版**: 基于原有模板,快速打印
  - **详细版**: 包含详细明细和汇总信息

### 4. 自动入账标记
- 生成销售清单后自动标记"是"
- 避免重复生成
