# 库存表自动化管理系统

一个基于Python的Excel库存表自动化工具,用于自动生成销售清单并管理出库记录。

## 📋 项目简介

本项目为东阳市欧亚金银丝有限公司开发的库存管理自动化工具,主要功能包括:

- ✅ 自动生成序号和日期
- ✅ 自动计算净重(毛重-除皮)
- ✅ 按客户和日期自动生成销售清单
- ✅ 支持两种销售清单模板(简单版和详细版)
- ✅ 自动标记入账状态

## 🚀 快速开始

### 方式一: 图形界面版本(推荐)

**无需Python环境,双击即可运行!**

1. 下载 `库存表管理系统.exe`
2. 双击运行程序
3. 选择输入文件(库存tmep.xlsx)
4. 点击"🚀 开始处理"
5. 查看生成的销售清单

**如何获取exe文件?**
- 在Windows电脑上执行打包(详见[打包说明.md](./%E6%89%93%E5%8C%85%E8%AF%B4%E6%98%8E.md))
- 或者从项目 Releases 页面下载

### 方式二: 命令行版本

**需要Python环境**

#### 环境要求
- Python 3.7+
- openpyxl 库

#### 安装依赖

```bash
pip install openpyxl
```

#### 使用方法

1. 将 `improve_inventory.py` 脚本放在与Excel文件同一目录
2. 确保Excel文件名为 `库存tmep.xlsx`
3. 运行脚本:

```bash
python improve_inventory.py
```

4. 脚本会生成 `库存_改进版.xlsx` 文件,包含所有销售清单

## 📊 功能特性

### 1. 自动序号
- 使用Excel公式自动生成连续序号
- 删除行时序号自动调整

### 2. 自动日期填充
- 运行脚本时自动填充空白日期为当天
- 格式: YYYY-MM-DD

### 3. 自动生成销售清单
- 按"出库日期 + 出库对象"分组
- 每组生成两个版本:
  - **简单版**: 基于原有模板,快速打印
  - **详细版**: 包含详细明细和汇总信息

### 4. 自动入账标记
- 生成销售清单后自动标记"是"
- 避免重复生成

## 💾 打包成可执行文件

项目支持打包成Windows可执行文件(.exe),无需Python环境即可运行。

### 快速打包(仅Windows)

1. 双击运行 `一键打包.bat`
2. 等待打包完成
3. 在 `dist` 目录下获取 `库存表管理系统.exe`

### 手动打包

```bash
# 安装依赖
pip install -r requirements.txt

# 执行打包
python build_exe.py
```

详细说明请查看 [打包说明.md](./%E6%89%93%E5%8C%85%E8%AF%B4%E6%98%8E.md)
